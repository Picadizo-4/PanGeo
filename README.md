# 🌍 PanGeo

**Trabajo de Fin de Grado (TFG)**

## 📱 Descripción del Proyecto
PanGeo es una aplicación móvil Android nativa orientada al aprendizaje gamificado de la geografía. A través de mapas interactivos, los jugadores pueden poner a prueba sus conocimientos identificando países de forma visual. 

El Producto Mínimo Viable (MVP) de este proyecto se centra en el mapa político de Europa.

## 🛠️ Tecnologías y Arquitectura
* **Lenguaje:** Kotlin
* **Arquitectura:** MVVM (Model-View-ViewModel)
* **Interfaz y Gráficos:** Mapas interactivos basados en Vector Drawables (SVG)
* **Base de Datos y Backend:** Firebase (Authentication y Cloud Firestore con estructura plana)
